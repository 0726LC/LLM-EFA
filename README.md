# LLM-EFA

Large Language Models for Short-Term Electricity Demand Forecasting under Abrupt Changes.

## Project Status

This repository is under active open-sourcing after paper submission.
To ensure transparent and reproducible release, we are publishing components in stages:

1. Public data interfaces and preprocessing pipeline.
2. Reproduction scripts for key paper results.
3. Full training/inference code and extended documentation.

Please check [TODO.md](./TODO.md) for the detailed roadmap and timeline.

## Current Scope

- Task: short-term electricity demand forecasting under abrupt changes.
- Planned inputs: weather, load history, calendar/event features, and related public signals.
- Planned outputs: reproducible benchmarks, model checkpoints/inference scripts, and paper-aligned experiment configs.
