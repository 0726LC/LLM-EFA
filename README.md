# LLM-EFA

Large Language Models for Short-Term Electricity Demand Forecasting under Abrupt Changes

## Status

This repository is being prepared as the public codebase for the submitted **LLM-EFA** paper. To make the release transparent and review-friendly, materials will be opened progressively rather than appearing all at once.

- **Available now:** repository status note, public release roadmap, and folder-level documentation skeletons.
- **Coming next:** public data documentation, download/preprocessing scripts, paper-aligned result artifacts, configs, and evaluation utilities.
- **After paper acceptance:** full model implementation, training pipeline, and complete experiment code.

## Release principles

- Prioritize reproducibility of public results.
- Clearly separate redistributable and non-redistributable assets.
- Keep paper-facing artifacts traceable to configs, scripts, and released result files.

## Planned repository structure

- `data/`: public data docs, download instructions, sample files, preprocessing notes.
- `results/`: paper-aligned key tables, figures, and exported metrics.
- `configs/`: experiment settings, seeds, and evaluation presets.
- `scripts/`: data preparation, evaluation, and reproduction utilities.
- `TODO.md`: public release checklist and roadmap.
